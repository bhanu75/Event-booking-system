# Event Booking System API

Backend API for event booking with role-based access control.

## Features
- User authentication (Organizers & Customers)
- Event management (CRUD)
- Ticket booking system
- Background job processing
- Analytics dashboard

## Tech Stack
- Node.js
- Express.js
- JWT Authentication
- Custom Job Queue
